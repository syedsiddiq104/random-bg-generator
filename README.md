# Random Background Color Generator

A simple and interactive web application that generates random background colors with a single click and displays the HEX color code.

##  Live Demo
https://syedsiddiq104.github.io/random-bg-generator/

##  Features
- Generates random HEX color codes
- Instantly updates background color
- Displays selected color code on screen
- Smooth button click animation
- Lightweight and fast

##  Tech Stack
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

##  How It Works
- A random number is generated using JavaScript
- The number is converted into a HEX color format
- The background color updates dynamically on button click
